# `07` removeProperty

## 📝 Instrucciones:

1. Escribe una función llamada `removeProperty`.

Dado un objeto y una key, `removeProperty` elimina esa llave de dicho objeto.

```Javascript
var obj = {
  name: 'Sam',
  age: 20
}
removeProperty(obj, 'name');
console.log(obj.name); // --> indefinifo
```
